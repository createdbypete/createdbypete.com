Removing Heroku Toolbelt

So you would like to uninstall the Heroku Toolbelt from OS X and you’ve noticed there is no information in the documentation. No worries, you can run the following commands in a Terminal prompt:

rm -rf ~/.heroku
sudo rm -rf /usr/local/heroku /usr/bin/heroku

This will delete the directories Heroku Toolbelt created during installation.
